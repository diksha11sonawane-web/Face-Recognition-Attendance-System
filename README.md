# 🎯 Real-Time Face Recognition Attendance System

## 📌 Overview
This project is an AI-powered attendance management system that uses computer vision and face recognition to automatically detect and recognize faces and mark attendance in real time.

## 🚀 Features
- Real-time face detection
- Face recognition
- Automatic attendance marking
- Attendance record management
- Duplicate attendance prevention

## 🛠 Technologies Used
- Python
- OpenCV
- Face Recognition
- NumPy
- Pandas
